Drop your .txt cookie files here inside the service folders:

  cookies/netflix/       ← Netflix cookies/accounts
  cookies/crunchyroll/   ← Crunchyroll cookies/accounts
  cookies/spotify/       ← Spotify cookies/accounts

Each .txt file = one account. Supported formats:
  - email:password
  - email:password:extra
  - email=x\npassword=y
  - Raw cookie string

Alternatively, send .txt files directly to the bot via Telegram (admin only).
The bot will ask which service the cookie belongs to.
